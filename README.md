## Description

> - A frontend application game that allows you to increase the score as you jump the ball. If the ball reach the bottom the score will reset to zero.

<br />

> - App: https://web-jumping-ball.vercel.app 

<br />
<br />
<br />



## Repository Setup
```plaintext
yarn install
yarn build
yarn dev
```
